# BulkIndexDocumentsRequest

Describes the request body of the /bulkindexdocuments API call

## Example Usage

```typescript
import { BulkIndexDocumentsRequest } from "@gleanwork/api-client/models/components";

let value: BulkIndexDocumentsRequest = {
  uploadId: "<id>",
  datasource: "<value>",
  documents: [
    {
      datasource: "<value>",
    },
  ],
};
```
