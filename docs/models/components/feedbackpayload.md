# FeedbackPayload

Optional payload for feedback reporting.

## Example Usage

```typescript
import { FeedbackPayload } from "@gleanwork/api-client/models/components";

let value: FeedbackPayload = {
  customJson:
    "{\"comment\": \"glean is awesome!\", \"sender\": \"happycustomer@customer.com\"}",
};
```
