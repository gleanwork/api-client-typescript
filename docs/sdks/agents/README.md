# Agents
(*agents*)

## Overview

### Available Operations

* [runagent](#runagent) - Runs an Agent.
* [listagents](#listagents) - Lists all agents.
* [getagentinputs](#getagentinputs) - Gets the inputs to an agent.

  bearerAuth: process.env["GLEAN_BEARER_AUTH"] ?? "",

    runAgentRequest: {},
