# Images
(*client.images*)

## Overview

* [get](#get) - Get image

## get

Serves images of various types (profile pic, background, UGC thumnail/content, etc).

### Example Usage

```typescript
import { Glean } from "@gleanwork/api-client";

const glean = new Glean({
  bearerAuth: process.env["GLEAN_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await glean.client.images.get({});

  // Handle the result
  console.log(result);
}

run();
```

## upload

Upload images for various types (profile pic, background, UGC thumnail/content, etc) with additional metadata.

### Example Usage

```typescript
import { Glean } from "@gleanwork/api-client";
import { openAsBlob } from "node:fs";

const glean = new Glean({
  bearerAuth: process.env["GLEAN_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await glean.client.images.upload({
    requestBody: await openAsBlob("example.file"),
  });

  // Handle the result
  console.log(result);
}

run();
```
