# Examples

## Run an example

```bash
# 1. cd into the root of the repo
cd api-client-typescript 

# 2. Install dependencies
npm install

# 3. Build the project
npm run build

# 4. cd into the examples directory
cd examples/

# 5. Fill in the .env file
cp .env.example .env

# 6. Install dependencies
npm install

# 7. Run the script
npm run script -- ./src/example.ts
```
